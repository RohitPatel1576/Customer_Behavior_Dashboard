# Customer Behavior Dashboard - Power BI

## Project Overview
This project is an interactive Customer Behavior Dashboard developed using Power BI. It helps analyze customer purchasing patterns, spending behavior, customer segmentation, and sales trends to generate business insights and improve customer engagement.

## Features
- Customer Purchase Analysis
- Customer Segmentation
- Spending Behavior Insights
- Sales Trend Analysis
- KPI Tracking Dashboard
- Interactive Filters & Visualizations

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Excel / CSV Dataset

## Key Insights
- Identified customer purchasing patterns
- Analyzed high-value customers
- Monitored customer spending trends
- Tracked sales performance and engagement
